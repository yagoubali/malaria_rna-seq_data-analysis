# Required tools
- trimmomatic v 0.39 from [THE USADEL LAB](http://www.usadellab.org/cms/?page=trimmomatic)

# Required files
- Adapters: TruSeq2, TruSeq3 ,  and  Nextera, from [This GitHub Repo](https://github.com/timflutre/trimmomatic)
- Adapters : Kapa adapter (AGATCGGAAGAGCAAGATCGGAAGAGC)
- The above adaters can be downloaded from [this link](adapters)

# Parameters
- trimmomatic  trimming parameters
```
ILLUMINACLIP:${adapters_dir}/${adapter}:2:30:10 LEADING:3 TRAILING:3 SLIDINGWINDOW:4:20 MINLEN:20
```

trim_PE(){

 adapter=$2

mkdir -p ${scratch_dir}/${Read}
